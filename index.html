Apache Amaterasu (Incubating) is an open source, configuration managment and deployment framework for data pipelines

Find Out More

A new approach

Amaterasu is a simple and powerful framework to build, configure and dispense data processing pipelines. It aims to help data engineers and data scientists to compose, configure, test, package, deploy and execute data pipelines written using multiple tools, languages and frameworks

Get Started!

Features:

Simple to code

Code your data apps using the languages and frameworks of your choice.

Configuration managment

Configure your apps and frameworks using a powerful YAML DSL for multiple environments.

Ship your code

Amaterasu uses Git for applications and configurations repositories

Easy to deploy

Deploy and run in a single line!
